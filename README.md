# Zenith - Mobile App Wireframing Case Study

A premium UI/UX Documentation and Case Study produced as part of the Thiranex internship program. This project demonstrates the full design thinking cycle for a mental wellness mobile application.

## 🚀 Key Features Explored
- **User Research**: In-depth analysis of user needs and digital wellness behaviors.
- **User Personas & Flows**: Strategic mapping of target users (Liam & Sarah) and their navigation journeys.
- **Figma/Adobe XD Blueprinting**: Low-fidelity structural wireframes designed with industry-standard grids.
- **Design Thinking Methodology**: A complete 5-stage documentation (Empathize, Define, Ideate, Prototype, Test).

## 🎨 Tech Stack & Tools
- **Frontend**: HTML5, CSS3 (Custom Design System), JavaScript (ES6+).
- **Design Style**: Premium Glassmorphism, Arctic Blue Palette.
- **Assets**: Professional 3D Avatars and AI-generated Artboards.

## 🔗 Live Preview
To view the interactive documentation, clone this repository and open `index.html` in any modern web browser.
